# Gjun

## 项目结构

```
assets/
  css/
    main.css                # CSS 入口文件（按顺序导入以下样式文件）
    base.css                # 变量、基础重置、通用工具类
    layout.css              # 全局布局模块（导航/主题切换/移动菜单等）
    sections.css            # 各页面区块样式
    responsive.css          # 响应式断点与设备适配样式
  js/
    main.js                 # 前端入口初始化
    modules/
      theme.js              # 主题切换逻辑
      particles.js          # 背景粒子动画
      nav.js                # 导航栏与移动端菜单交互
      interactions.js       # 滚动动效、数字动画、平滑滚动
      cert-marquee.js       # 移动端证书滚动逻辑
index.html                  # 页面结构文件（不含内联样式/脚本）
```
